📌 **Project Overview**

Diabetic Nephropathy is a serious kidney disease caused by long-term diabetes. Early detection is important to prevent kidney failure and improve patient care. This project proposes an automated deep learning system to detect diabetic nephropathy using patient medical data and artificial neural networks.

The system analyzes clinical parameters and predicts whether a patient is at risk of diabetic nephropathy. This helps doctors make faster and more accurate decisions.

🎯 **Objectives**
Detect diabetic nephropathy at an early stage
Improve prediction accuracy using deep learning
Reduce manual diagnosis effort
Assist healthcare professionals in decision-making
